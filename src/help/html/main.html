SISTEMA PARA CONTROL DE PACIENTES

Esta ventana resolverá las dudas que se puedan tener acerca del "Sistema para el control de pacientes".
Versión2.0 Realizado en lenguaje JAVA V_1.7.0_79
Servidor MySQL: MariaDB 10.1.13

Esta página es la ayuda principal. La aplicación consta de una ventana principal y en su interior se ejecuta cada opción elegida por el usuario con ayuda del menú que se encuentra en la parte superior del sistema.

A continuación se muestran las ventanas e iconos principales que componen el sistema.
Se ofrece en la parte izquierda de esta ventana un menú en el cual cada opción despliega una sub ventana para explicar la funcionalidad de cada icono-ventana con más detalle.

  • 2. Ventana de inicio.



  • 3. Menú principal, en el que cada opción da paso a una ventana diferente



    A continuación se dará una breve descripción de cada opción del menú principal

  • A) Icono que da paso a la ventana para dar de alta un nuevo paciente o ver la información de uno ya existente.
  • B) Icono para consultar las próximas citas médicas por fecha.
  • C) Icono para realizar una nueva consulta médica.
  • D) Icono para realizar una nueva receta médica.
  • E) Icono para dar de alta un nuevo medicamento, tratamiento, estado civil, familiares o precios.
  • F) Icono para consultar o dar de alta los datos de Antecedentes Heredofamiliar del paciente.
  • G) Icono para la realización o consulta de pagos.
  • H) Icono para abrir la ventana de chat para comunicación con otro usuario del sistema.
  • I) Icono para abrir la ventana de ayuda para usuarios del sistema.
  • J) Icono de salida de usuario.
  • K) Icono para la limpieza de ventana principal.
